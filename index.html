Instal Termux di Android

Perintah instalasi termux:

  • $ apt update && apt upgrade
  • $ termux-setup-storage

Pertama, download aplikasi termux di Google Play Store.

Setelah terinstal, buka aplikasi termuk dan masukan perintah:

$ apt update && apt upgrade

Copy dan paste di termux perintah di atas (Hilangkan $) untuk mengupdate dan upgrade paket yang ada di termux.

Biasanya akan muncul prompt, untuk melanjutkan ketik Y atau n untuk membatalkan.

Ketik Y untuk melanjutkan dan n untuk membatalkan

Jika Hp Android kamu Android 6.0 marshmallow ke atas maka masukan perintah:

$ termux-setup-storage

Karena sa’at ini termux belum bisa mengakses media penyimpanan di Android 6.0 ke atas.

Selamat, termux sudah berhasil di instal di hp Android kamu.

Bagaimana cara menggunakan termux?

Jika Anda baru pertama menggunakan termuk (belum pernah menggunakan linux) pasti bingung kan bagaimana cara menggunaka aplikasi termux ini.

Berbeda dengan aplikasi-aplikasi Android pada umumnya, jika di aplikasi lainnya ada pilihan menu untuk menjalankannya. Tapi kalau di termux kita harus memasukan perintah (comand termux).

Jadi, kamu setidaknya kamu harus tahu beberapa perintah dasar termux.

Basic Comand Termux

Berikut ini adalah perintah dasar Termux yang harus kamu hafalin:

apt install Adalah perintah termux untuk menginstal.

  • Penggunaan: apt install [options]
  • Contoh: apt install python

cd perintah termux untuk mengakses penyimpanan. (folder terbuka)

  • Penggunaan: cd
  • Contoh: cd / sdcard

chmod Adalah perintah termux untuk mengubah izin akses file atau direktori.

  • Penggunaan: chmod [opsi]
  • Contoh: chmod 777 python.py

cp -f Adalah perintah untuk menyalin file.

  • Penggunaan: cp-f [opsi]
  • Contoh: cp -f filename.php filename (folder yang akan ditempelkan)

cp -rf Adalah perintah untuk menyalin folder.

  • Penggunaan: cp-rf [opsi]
  • Contoh: cp -rf folder1 folder2 (folder mana yang akan ditekan)

mkdir Adalah perintah untuk membuat folder baru.

  • Penggunaan: mkdir [opsi]
  • Contoh: mkdir new folder

exit Tentu saja untuk keluar dari aplikasi thermux. Nama ini juga keluar

  • Penggunaan: exit
  • Contoh: exit

ls Adalah perintah untuk menampilkan file, filder, script dan aplikasi / software yang terinstal di termux.

  • Penggunaan: ls
  • Contoh: Untuk mengetahui apa yang telah Anda instal pada termux ketil ls

mv Adalah perintah untuk memindahkan file / folder. dan ganti nama file / folder

  • Penggunaan: mv [opsi]
  • Contoh: mv filename.php filename (untuk memindahkan file)
  • mv filelama.php filebaru.php (untuk mengubah nama file)

rm -f Adalah perintah untuk menghapus file.

  • Penggunaan: rm-f [opsi]
  • Contoh: rm -f filename.php

apt list Perintah termux adalah untuk melihat daftar skrip yang diinstal.

  • Penggunaan: apt list
  • Contoh: apt list

apt update Adalah perintah termux untuk memperbarui paket atau skrip yang telah diinstal.

  • Penggunaan: apt update
  • Contoh: apt update

apt upgrade Adalah perintah termux untuk memperbarui paket yang ada ke versi terbaru.

  • Penggunaan: apt upgrade
  • Contoh: apt upgrade

clear Adalah perintah termux untuk menghapus (membersihkan) layar pada terminal

  • Penggunaan: clear
  • Contoh: clear

Itu adalah beberapa perintah termux dasar yang harus Anda hafal. ada banyak perintah thermux dasar lainnya. jadi, teruslah bersemangat tentang belajar menggunakan thermux.

Instal Metasploit di Termuk

Berikut ini adalah perintah untuk menginsat tool metaspoit di termux:

  • $ pkg install upgrade
  • $ pkg install git
  • $ git clone https://github.com/Hax4us/Metasploit_termux

Cara Menggunakan Metasploit

Untuk menjalankan metasploit di termux kamu cukup masukan perintah-perintah di bawah ini:

  • $ ls
  • $ cd Metasploit_termux
  • $ chmod +x metasploit.sh
  • $ ./metasploit.sh
